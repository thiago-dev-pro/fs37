# MARKDOWN
Markdown é uma linguagem de marcação leve e simples.

# TÍTULOS
Podemos adicionar títulos ao markdown.

# TÍTULO 1
## TÍTULO 2
### TÍTULO 3
#### TÍTULO 4
##### TÍTULO 5
###### TÍTULO 6

# ÊNFASE NO TEXTO
Negrito: usa-se **dois pares de asteriscos** \*\*negrito\*\*  
Itálico: usa-se *um par de asteriscos* \*itálico\*  
Negrito e Itálico: usa-se ***três pares de asteriscos*** \*\*\*itálico\*\*\*  
Texto riscado: usa-se ~~texto~~
